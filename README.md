# Chrome Word Sentence Recorder

一个用于英文阅读的 Chrome 扩展。  
在网页中**单击英文单词**，即可自动保存这个单词及其所在的**完整句子**，并生成可复习的**单词卡**。

适合阅读英文文章、新闻、技术文档、论文、博客时边读边积累词汇和例句。

---

## 功能特色

- **单击即保存**
  - 在网页中单击英文单词，自动记录该单词和所在完整句子
- **同单词自动合并**
  - 同一个单词只保留一张词卡，不会重复收录
- **新句子自动追加**
  - 如果同一个单词出现在新的句子中，会自动新增为新的例句
- **完整上下文记录**
  - 保存单词、完整句子、页面标题、页面 URL、保存时间
- **单词卡复习**
  - 支持以卡片形式查看单词和例句，便于回顾和复习
- **本地存储**
  - 数据保存在浏览器本地，不依赖外部服务器

---

## 适用场景

- 阅读英文新闻时积累生词
- 阅读技术文档时保存真实语境
- 在日常网页浏览中随手建立个人单词本
- 基于例句进行复习，而不是只背孤立单词

## 安装方式

### 方法一：本地加载（开发者模式）

1. 下载项目源码并解压
2. 打开 Chrome，进入：

   `chrome://extensions/`

3. 打开右上角 **开发者模式**
4. 点击 **加载已解压的扩展程序**
5. 选择项目文件夹

安装完成后，即可在浏览器工具栏看到扩展图标。

---

## 使用方法

1. 打开任意英文网页
2. **单击一个英文单词**
3. 插件自动保存该单词及其所在完整句子
4. 点击浏览器右上角扩展图标
5. 在弹窗中查看记录或进入单词卡模式复习

### 自动处理规则

- 如果是**新单词**：新建一张词卡
- 如果是**已有单词，但句子不同**：追加为新例句
- 如果是**已有单词，且句子相同**：不重复保存

## 数据结构说明

每个单词对应一张词卡，词卡下可包含多条例句。

示例：

```json
{
  "word": "capture",
  "examples": [
    {
      "sentence": "The tool can capture the full sentence automatically.",
      "title": "Example Page",
      "url": "https://example.com",
      "savedAt": "2026-03-08T10:00:00.000Z"
    },
    {
      "sentence": "This extension helps capture words while reading.",
      "title": "Another Page",
      "url": "https://example.org",
      "savedAt": "2026-03-08T10:05:00.000Z"
    }
  ]
}
